# Как работает AI Video Pipeline

Полностью автоматический конвейер: от текстового сценария до готового вертикального ролика (9:16), опубликованного на 5 платформах. Ручной монтаж не требуется.

---

## Общая схема

```
Сценарий (YAML)
    │
    ▼
┌─────────────────────────────────────────────────────┐
│                    PIPELINE                          │
│                                                      │
│   1. Картинки ──► 2. Видео ──► 3. Озвучка           │
│                                     │                │
│   4. Субтитры ◄─────────────────────┘                │
│        │                                             │
│        ▼                                             │
│   5. Сборка (FFmpeg)                                 │
│        │                                             │
│        ▼                                             │
│   6. Публикация                                      │
│      ├── Telegram                                    │
│      ├── Instagram Reels                             │
│      ├── YouTube Shorts                              │
│      ├── VKontakte                                   │
│      └── TikTok                                      │
└─────────────────────────────────────────────────────┘
    │
    ▼
Готовый ролик (MP4, 1080x1920)
```

---

## Этапы подробно

### 1. Генерация изображений

**Что происходит:** Для каждой сцены из сценария генерируется статичная картинка.

**Как:** Текстовый промпт отправляется через API на [Replicate.com](https://replicate.com), где запускается модель генерации изображений (FLUX, SDXL и др.). Модель возвращает PNG 1080x1920 пикселей (вертикальный формат).

**Пример:**
```
Промпт: "Cute cartoon strawberry family in a cozy kitchen,
         Pixar style, warm lighting, 3D render"

Модель: FLUX 1.1 Dev ($0.025 за картинку)

Результат: scene_01.png (1080x1920)
```

**Модели на выбор:**

| Модель | Цена | Качество |
|--------|------|----------|
| FLUX 1.1 Pro | $0.04 | Максимальное |
| FLUX 1.1 Dev | $0.025 | Высокое |
| FLUX Schnell | $0.003 | Хорошее (самая быстрая) |
| SDXL | $0.01 | Среднее |

---

### 2. Генерация видео (Image-to-Video)

**Что происходит:** Статичная картинка "оживает" — превращается в 5-секундный видеоклип.

**Как:** Картинка + текстовый промпт с описанием движения отправляются через Replicate API в модель image-to-video. Модель анимирует изображение: камера двигается, персонажи шевелятся, добавляются эффекты. Возвращается MP4.

**Пример:**
```
Вход:   scene_01.png + "camera slowly zooms in, characters talking"
Модель: Hailuo 2.3 ($0.30 за клип)
Выход:  scene_01.mp4 (5 сек, ~24fps)
```

**Модели на выбор:**

| Модель | Цена | Особенности |
|--------|------|-------------|
| Hailuo 2.3 | $0.30 | Лучшее качество (default) |
| Hailuo 2.3 Fast | $0.12 | Бюджетный вариант |
| Kling v2.5 | $0.50 | Самое плавное движение |
| Wan 2.2 Fast | $0.09 | Самый дешёвый |
| Veo 3.1 | $6.00 | Лучшее качество (дорого) |

---

### 3. Озвучка (Text-to-Speech)

**Что происходит:** Текст озвучки из сценария превращается в аудиофайл с голосом.

**Как:** Есть два варианта:

- **Edge TTS (бесплатно)** — Microsoft Neural Voices. Работает через API Microsoft, поддерживает русские голоса. Качество хорошее, стоимость нулевая.
- **Replicate TTS (платно)** — модели с клонированием голоса (Chatterbox). Более естественное звучание, поддержка кастомных голосов.

**Пример:**
```
Текст:  "Папа Клубничка пришёл домой и увидел, что на кухне..."
Голос:  ru-RU-DmitryNeural (Edge TTS, бесплатно)
Выход:  scene_01.mp3 (~7 секунд)
```

---

### 4. Субтитры

**Что происходит:** Создаётся файл субтитров (SRT) с точными таймингами.

**Как:** Это **не API-вызов** — код сам генерирует субтитры:
1. Берёт текст озвучки каждой сцены
2. Определяет длительность аудиофайла
3. Разбивает текст на строки (макс. 35 символов)
4. Равномерно распределяет строки по длительности аудио
5. Записывает SRT-файл с временными метками

**Пример SRT:**
```srt
1
00:00:00,000 --> 00:00:03,500
Папа Клубничка пришёл
домой и увидел,

2
00:00:03,500 --> 00:00:07,000
что на кухне происходит
что-то странное...
```

---

### 5. Сборка (Assembly) — автоматический монтаж

**Это ключевой этап. Весь монтаж выполняется автоматически через FFmpeg — без ручного вмешательства.**

FFmpeg — это консольный инструмент для обработки видео, установленный на Mac. Python-код формирует FFmpeg-команды и запускает их.

#### Шаг 5.1: Синхронизация видео и аудио

Для каждой сцены видеоклип подгоняется под длительность озвучки:

```
Видео: 5 сек (от Replicate)
Аудио: 7 сек (от TTS)

Решение: замедлить видео в 1.4 раза (7/5 = 1.4)
          setpts=1.4*PTS → видео теперь тоже 7 сек
```

Также нормализуется разрешение (всё приводится к 1080x1920, 30fps) и заменяется оригинальный звук клипа на озвучку.

#### Шаг 5.2: Склейка сцен

Все подготовленные сегменты склеиваются друг за другом через FFmpeg concat:

```
scene_01 (7 сек) + scene_02 (6 сек) + scene_03 (8 сек) + scene_04 (5 сек)
= итоговое видео (26 сек)
```

#### Шаг 5.3: Фоновая музыка (опционально)

Если указан музыкальный файл:
- Зацикливается на всю длину видео
- Микшируется с озвучкой на 15% громкости
- Fade-in 2 сек в начале, fade-out 2 сек в конце

#### Шаг 5.4: Вшивание субтитров

SRT-файл рендерится прямо на видеокадры через FFmpeg subtitle filter. Настраивается шрифт, цвет, обводка, позиция (внизу экрана по умолчанию).

**Результат:** один финальный MP4-файл, готовый к публикации.

---

### 6. Публикация

**Что происходит:** Готовый ролик автоматически выкладывается на выбранные платформы.

**Как:** Для каждой платформы используется свой SDK/API:

| Платформа | Библиотека | Метод |
|-----------|-----------|-------|
| Telegram | python-telegram-bot | Отправка видео в канал через Bot API |
| Instagram | instagrapi | Загрузка Reels через unofficial API |
| YouTube | google-api-python-client | YouTube Data API v3 (OAuth2) |
| VKontakte | vk-api | Загрузка видео через VK API |
| TikTok | TikTokApi | Загрузка через API (+ ручной fallback) |

К каждому посту добавляются: заголовок, описание, хештеги (настраиваемые в конфигурации).

## Стоимость одного ролика

Пример для 4 сцен:

| Пресет | Картинки | Видео | Озвучка | Итого |
|--------|----------|-------|---------|-------|
| Premium | FLUX Pro ($0.16) | Hailuo 2.3 ($1.20) | Edge TTS (бесплатно) | **~$1.36** |
| Standard | FLUX Dev ($0.10) | Hailuo Fast ($0.48) | Edge TTS (бесплатно) | **~$0.58** |
| Budget | FLUX Schnell ($0.01) | Wan Fast ($0.36) | Edge TTS (бесплатно) | **~$0.37** |

Сборка и субтитры — бесплатно (локальный FFmpeg).
Публикация — бесплатно (API платформ).

## Типы контента из коробки

### Семейка Ягодок (Fruit Soap Opera)
Мыльная опера с антропоморфными фруктами/овощами. Трендовый формат русского Instagram.
- 6 персонажей (Папа Клубничка, Мама Клубничка, Дядя Огурец и др.)
- 5 готовых сюжетов (тайна на кухне, новый сосед, школьные проблемы и др.)

### Character Remix
Известные персонажи в вирусных контекстах.
- 5 персонажей (Шрек, Эльза, Губка Боб, Чебурашка, Маша)
- 3 контекста (запись рэп-альбома, курьер доставки, фитнес-блогер)

### Бизнес-маскоты (Business Mascot)
Контент для продвижения бизнесов с кастомными маскотами.
- 3 типа бизнеса (недвижимость, автосервис, кофейня)
- Уникальный маскот для каждого (Дом Домыч, Гаечка, Бариста Бинни)
